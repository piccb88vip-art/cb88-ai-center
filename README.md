# CLICKBET88 AI Center V5

Project GitHub Pages untuk landing + AI boot loading + dashboard prediksi CLICKBET88.

## Cara upload
1. Upload seluruh isi folder ini ke repository GitHub `cb88-ai-center`.
2. Pastikan struktur root berisi `index.html`, folder `css`, dan folder `js`.
3. Settings > Pages > Deploy from branch > main > /(root).
4. Buka: https://piccb88vip-art.github.io/cb88-ai-center/

## Setting
Edit `js/config.js` untuk mengganti:
- `sheetApi`
- `loginUrl`
- logo/background

## Catatan
Kalau `sheetApi` kosong, website memakai data fallback contoh.
